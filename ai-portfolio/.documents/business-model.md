# AI Portfolio App - Business Model

Last updated: 2026-02-20

## Purpose
Build a recruiter-facing AI portfolio app that answers fit questions quickly and converts visitors into qualified contact conversations.

## Core Outcomes
- Increase interview requests from recruiter/hiring manager traffic.
- Improve signal quality by giving grounded, role-relevant AI answers.
- Provide a clear contact path from chat interactions.

## Operating Workflow
1. Create tickets in `.documents/.tickets/current/`.
2. Plan work in `.documents/.plans/pending/`.
3. Implement and verify changes in `ai-portfolio`.
4. Record release notes in `.documents/CHANGELOG.md`.
