# Career Portfolio Project Plan

## Goal
Ship a fast, credible portfolio site that supports job applications and outreach.

## Core Outcomes
- Clear value proposition above the fold.
- Strong experience proof through projects and measurable results.
- Smooth navigation and interaction on desktop and mobile.

## Delivery Approach
1. Create scoped tickets under `.documents/.tickets/current/`.
2. Create phased plans under `.documents/.plans/pending/`.
3. Implement and manually verify each plan before commit/PR.
